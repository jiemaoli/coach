# 英文单词本

| 单词/短语 | 中文意思 | 项目交易语境用法 |
|-----------|----------|------------------|
| rank beginner | 彻头彻尾的初学者 | rank 作形容词表示"完全的、十足的"，带贬义 |
| SIM | 模拟交易 | Simulation 的缩写，本项目中指模拟交易。如：trade on SIM only = 只用模拟交易练习 |
| eagerness | 急切、急于交易 | 交易心理中的负面心态，与 patience（耐心）相对，容易导致过早入场。如：Eagerness and impatience are signs of a losing inexperienced trader |
| unicycle | 独轮车 | 文中比喻交易需要同时掌握多种技能，如同骑独轮车抛刀。如：juggling 4 flaming knives while riding a unicycle |
| juggle | 杂耍、抛接 | 文中比喻同时处理多项交易任务。如：juggle only one flaming knife = 先只做一种交易（A2） |
| consistent | 稳定的、持续的 | 交易语境中指结果可重复、不忽好忽坏。如：consistent trader = 稳定盈利的交易者；your consistency will only improve when your outlook changes |
| transition | 市场状态转换 | 非"交易"。市场三种方向（up/down/flat）之间的转换，3×3=9 种，即博客名 Nine Transitions 的由来 |
| flatten out / flatten | 走平；平仓 | 市场 flatten out = 趋势转横盘；flatten position = 平掉仓位归零。如：when a trending market flattens out, you need to exit |
| slightly | 略微、稍微 | 交易语境中强调程度小但有实际影响。如：slightly above EMA = 略微高于均线；slightly better entry = 略微更好的入场价；slightly larger stops = 略微放宽止损 |
| overshoot | 突破/超越（趋势通道线） | 核心术语。TCL overshoot = 突破趋势通道线；obnoxious overshoot = 过度突破，强反转信号。如：overshoot and reversal bar = 突破后出现反转K线；overshoot only imply a pullback is due = 突破只暗示会有回调，obnoxious overshoot才预示真正反转 |
| TCL | Trend Channel Line 趋势通道线 | 用于找出场点。TCL overshoot (TCL OS) = 突破趋势通道线；TCL failure = 突破后未反转。如：TCL typically indicates favorable location for exit = TCL通常是理想的出场位置 |
| overlap | 重叠（K线价格区域） | overlapping bars = 交易区间；信号K线有overlap是差信号，应避免交易。如：never buy above or sell below overlaps；overlaps produce poor signals；Any overlap practically guarantees the trading range attribute of the signal bars |
| fade | 反向交易 | counter-trend trading。交易区间内fade breakouts；趋势中不要fade。如：fade breakouts in a trading range；don't fade a trend till after a trendline break；"fading is extremely hard"；Repeatedly trying to fade a strong trend is the surest way to empty your trading account |
| W1P | Wedge 1st Pullback 楔形反转后首次回撤 | 四大核心交易之一（A2/W1P/DP/fBO）；比直接交易W更安全；目标通常可回撤整个楔形；失败则变A2空头。如：W1P is a much better trade than W；W1Ps often retrace the entire wedge；A failed W1P is also an A2 short on failure |
| A2 | 2 Legged move to Average 2腿回撤到均线 | 四大核心交易之首；新手唯一推荐交易。趋势中2腿回撤接近EMA入场；失败可反手。如：start with A2 and stick to A2 only；A2 is my favorite entry；A failed A2 long becomes an A2 short；The first A2 of the day is very likely to be successful |
| imply | 暗示、意味着 | 交易语境中表示间接暗示。如：imply more down = 暗示更多下跌；imply reversal = 暗示反转；do not imply reversal = 不意味着反转；imply a trend move = 暗示趋势移动。注意区分：imply（间接暗示）≠ mean（直接意味着） |
| extreme behavior | 极端行为 | 判断反转是否可靠的核心概念。True wedges represent extreme behaviors = 真正的楔形代表极端行为，几乎不会失败；W are extreme behaviors；通道中小幅突破 hardly a new push or extreme behavior = 不算极端行为；没有extreme behavior的反转不可靠，应等W1P |
| impending | 即将到来的、迫在眉睫的 | 描述市场已出现早期信号，某个重大变化马上发生。impending news = 即将公布的新闻；impending reversal = 即将到来的反转；impending crash = 即将到来的崩盘；impending breakouts = 即将发生的突破 |
| DP | Double top/bottom and Pullback 双顶/双底回撤 | 四大核心交易之一（A2/W1P/DP/fBO）；趋势生成器；本质是失败突破（fBO）。双底后小幅回撤不触底=做多；双顶后小幅回撤不触顶=做空。需在交易区间正确一端交易；区间太窄（≤4点）时不适用。如：DPs are often trend generators；DP should not be taken if its at the wrong end of the TR |
| exhaustive | 详尽的、全面的、彻底的 | 覆盖所有细节、毫无遗漏。如：An exhaustive description of all the above require their own blog posts = 以上内容的详尽描述需要各自专门的博文 |
