# 英文单词本

| 单词/短语 | 中文意思 | 备注 |
|-----------|----------|------|
| rank beginner | 彻头彻尾的初学者 | rank 作形容词表示"完全的、十足的"，带贬义 |
| SIM | 模拟交易 | Simulation 的缩写，本项目中指模拟交易 |
| eagerness | 急切、急于交易 | 交易心理中的负面心态，与 patience（耐心）相对，容易导致过早入场 |
| unicycle | 独轮车 | 文中比喻交易需要同时掌握多种技能，如同骑独轮车抛刀 |
| juggle | 杂耍、抛接 | 文中比喻同时处理多项交易任务 |
